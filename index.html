Viewer Notes

This page is only for validating the basemap layer. The public parking overlays continue to live in the main dashboard.

  • Basemap source: OSM-derived PMTiles
  • Renderer: Leaflet with PMTiles / Protomaps compatible runtime
  • Fallback: OSM raster tiles if PMTiles is not configured
